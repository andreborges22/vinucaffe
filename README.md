☕ Vinu Caffé — Sistema de Gerenciamento de Pedidos

Sistema desenvolvido para a cafeteria Vinu Café, com o objetivo de modernizar o atendimento através de pedidos digitais via QR Code, controle de pedidos em tempo real e organização operacional da cozinha.

🎯 Objetivo do Sistema

Reduzir a demora no atendimento presencial e melhorar a experiência do cliente por meio da digitalização completa do fluxo de pedidos.

O sistema substitui processos manuais por uma solução centralizada e automatizada.

⚙️ Funcionalidades
🔐 Autenticação
Login de usuários
Controle de acesso por perfil (cliente, atendente, admin)

🍽️ Gestão de Cardápio
Cadastro de itens do cardápio
Listagem de produtos disponíveis
Edição e atualização de produtos
Controle centralizado do menu

🧾 Pedidos
Realização de pedidos via sistema (QR Code)
Registro automático de pedidos
Controle de status do pedido
Organização entre atendimento e cozinha

👨‍🍳 Comanda / Cozinha
Atualização de status:
Em preparo
Pronto
Entregue
Visualização em tempo real dos pedidos
Fluxo organizado para produção

Tecnologias Utilizadas:
- Python
- Django REST Framework
- MySQL
- React
